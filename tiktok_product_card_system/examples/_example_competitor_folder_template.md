# Example Competitor Folder Template

## 文件用途
说明中文竞品资料文件夹的推荐输入结构。

## 中文结构
```text
竞品资料/
  某某产品竞品1/
    商品标题.txt
    SKU名称.txt
    商品详细信息.txt

    主图/
      主图1.png
      主图2.png
      主图3.png
      主图4.png
      主图5.png
      主图6.png

    详情图/
      详情图1.png
      详情图2.png
      详情图3.png
      详情图4.png
      详情图5.png

    产品参考图/
      参考图1.png
      参考图2.png
      参考图3.png
```

## 基础说明
- 文件夹名可以全中文。
- 图片名可以全中文。
- 用户只会标注主图1、主图2、详情图1、详情图2。
- 用户不会详细解释每张图是什么内容。
- 系统需要根据图片内容自行分析。
- 如果当前环境不能识图，必须诚实标记，不要假装看懂。
