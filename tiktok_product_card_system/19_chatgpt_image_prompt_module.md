# 19 ChatGPT Image Prompt Module

## 文件用途
建立面向 ChatGPT 生图的提示词规划模块。

## 基础规则
- 优先适配 ChatGPT 生图。
- 用户会上传产品参考图。
- 每一张主图单独输出提示词。
- 每一张详情图单独输出提示词。
- 提示词需要锁定产品结构、颜色、数量、材质、角度、场景、排版。
- 主图提示词和详情图提示词分开。
- 如果需要图片文字，必须指定英文短文案、位置和层级。
- 如果不需要文字，必须明确 no readable text / no typography。
- 不允许所有图片合并成一条大提示词。

## 第二阶段要填充的内容
补充提示词字段、负面约束和主图/详情图差异规则。

## 与其他文件的关系
接收主图规划、详情图规划、产品参考图和图片分析结果，输出到固定提示词模板。

## 竞品验证后的 ChatGPT 生图提示词规则（2026-07-08）
- 每一张主图、每一张详情图单独一条提示词；不得把整套图片合并成一条。
- 每条提示词必须包含：产品参考图垫图、产品结构锁定、颜色锁定、数量锁定、配件锁定、主视觉位置、文字位置、信息层级、移动端可读性、图片比例、背景、禁止改变项。
- 效果型主图要锁定“效果正在发生”：水束、泡泡、灯效、轨迹或完成后的可玩场景。
- 功能详情图要锁定证明方式：按钮特写、箭头轨迹、配件平铺、尺寸线、参数卡、步骤小图。
- 英文短文案必须原生化且短：One-Button Blast、Long-Range Spray、Bright LED Effects、Kit Includes、Easy Setup。
