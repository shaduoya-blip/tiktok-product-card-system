# TikTok Product Card System


## ChatGPT GitHub 应用读取入口

ChatGPT 通过 GitHub 应用读取本仓库时，应优先打开 `00_CHATGPT_READ_FIRST.md`。该文件是统一入口，说明正式知识源、必读顺序、按任务选择性读取规则、产品组索引和固定运行流程。


## ChatGPT 运行入口与运行包

ChatGPT 通过 GitHub 调用仓库时：

1. 先读取 `00_CHATGPT_READ_FIRST.md`。
2. 再读取 `CHATGPT_RUNTIME_BUNDLE.md`。
3. 根据当前产品选择性读取产品组、产品档案和专项文件。

`CHATGPT_RUNTIME_BUNDLE.md` 是日常运行精简包，不替代完整专项规则文件。

## 文件用途
这是一个长期复用的 ChatGPT Project 文件系统，用于 TikTok Shop US 商品卡规划。

系统未来支持：
- 产品信息输入
- 竞品商品卡输入
- 中文竞品资料文件夹
- 竞品主图/详情图图片分析
- 商品最大卖点提取
- 商品卡规划
- ChatGPT 生图提示词输出
